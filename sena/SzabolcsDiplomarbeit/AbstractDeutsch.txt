Event Cloud ist ein System das dem Benutzer eine historische Ansicht von gesammelten Eventstreams, welche von Senactives InTime Sense and Response Architecture erzeugt wurden, ermöglicht. InTime besitzt die Fähigkeit Events bezüglich vordefinierter Regeln zu korrelieren und liefert dadurch eine kausale Verfolgung von zusammenhängenden Events. Event Clouds Arbeit ist dabei diese Events zu sammeln und einen Volltextindex über diese zu legen um eine Google ähnliche Suche zu bieten. Des weiteren verfügt es über Werkzeuge die es erlauben verschiedene Aspekte von Geschäftsprozessen, die auf Eventkorrelationen basieren, zu untersuchen. Event Cloud ist über verschiedenen Stufen hinweg gewachsen um eine geeignete Architektur zu finden damit man einen Grundstein für spätere Arbeiten im Breich von Eventmining und Korrelierungsfindung legt.
